<objective>
Instrument PracticeScreen to record a completed session to IndexedDB when the celebration screen is reached. After writing the Session, immediately recalculate and upsert the TopicProgress record for that topic by aggregating all sessions.

Purpose: Provides the accuracy data that Phase 5's adaptive ordering (Plan 03) and parent dashboard (Plan 03) will consume. Without this write path, those features have no data.

Output: PracticeScreen emits one Session + one TopicProgress upsert per completed practice session. Tests confirm the write happens on celebration and does not happen on mid-session quit.
</objective>
